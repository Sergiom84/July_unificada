# Lucy Lara

Proyecto finalizado de negocio local de estetica con captacion, catalogo, reserva, chatbot y contenido SEO por servicios.

## Stack

React 18, TypeScript, Vite, Tailwind, Wouter, Express, Drizzle, OpenAI, Supabase RPC para conocimiento del chatbot y Render.

## Valor como plantilla

Plantilla avanzada para negocio local con IA/chatbot.

## Riesgos reutilizables

- Datos de negocio repetidos en varias capas.
- Reserva y backend pueden desincronizarse.
- Persistencia real no queda clara segun el analisis.
- Endpoints admin sin autenticacion segun el analisis.
- Al clonar hay que revisar dominio, email, telefono, horarios, URLs, CSP, SEO, chatbot y correo.
