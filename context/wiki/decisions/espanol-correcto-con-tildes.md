# Español correcto con tildes

## Contexto

La memoria de `Mente_unificada` está escrita para Sergio y se usa como base de procedimientos. Eliminar tildes por comodidad técnica reduce la calidad del contenido y transmite descuido.

## Decisión

Los documentos de procedimiento, wiki y prompts deben escribirse en español correcto, con tildes, `ñ`, signos de apertura y ortografía cuidada.

## Excepciones

No se deben modificar por estética:

- rutas de archivos;
- comandos;
- identificadores de código;
- nombres técnicos reales;
- claves de configuración;
- valores que deban permanecer tal como los usa una herramienta.

## Consecuencia práctica

Codex debe priorizar legibilidad y corrección lingüística en esta base de conocimiento. La regla de ASCII solo aplica cuando el contexto técnico lo justifica.
