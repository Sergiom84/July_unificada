# Analisis inicial de webs de cliente

## Resumen

Analisis comparativo de tres proyectos de Sergio: Lucy Lara, Zaidy y MHK Studio. El documento extrae patrones comunes, riesgos reutilizables y notas para convertir en memoria.

## Proyectos revisados

- [[Lucy Lara]]: plantilla avanzada para negocio local con IA, chatbot, reserva, catalogo y backend.
- [[Zaidy]]: plantilla estatica rapida para marca personal con blog, tienda, recursos y lead magnet.
- [[MHK Studio]]: plantilla moderna para web estatica con Astro y SEO local por ciudades.

## Patrones detectados

- Las webs de cliente estan orientadas a captacion.
- Identidad y contacto deben centralizarse.
- SEO basico es siempre necesario.
- Los formularios son un riesgo frecuente.
- Render aparece como plataforma de deploy recurrente.
- La IA/chatbot solo conviene cuando aporta valor real.
